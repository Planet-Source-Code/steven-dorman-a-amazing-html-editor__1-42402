Cool Tips and Tricks with HTML  

Below are some HTML tips and tricks that can make any old boring site look cool again. These directions have been created so even the worst of HTML programmers can most likely accomplish these tips and tricks. There are only 12 tips currently more will be added in soon at the Home Grown Productions website.

TRICKS
SOURCE CODE
EXAMPLE
Have your words scroll horizontally along the page.
<Marquee> WORDS YOU WANT TO SCROLL</Marquee>
 WORDS YOU WANT TO SCROLL  
Have an audio file play while users visit the page. In the source code you change bgsound.mid to the sound file you would like it to play.
<embed src="bgsound.mid" hidden="true" autostart="true" loop="1">
<noembed>Your browser doesn't support EMBED, but you can still listen to the background sound of this page by<a href="bgsound.mid"> clicking here.</a></noembed>
No Example
How to increase the traffic to your website with META tags.The tag in the source code should be put in <HEAD> </HEAD>. In the source code for description put a short description of your webpage. In keywords type in some keywords that relate to your site that the person might type in too see your site on the search engine. You can seperate Each Keyword by using a comma in between words.
 <meta name="description" content="How to promote your web site.">
<meta name="keywords" content="announcing web site, web submission, web promote, internet promotions, submit">
No Example
Have a image and then words next to the picture. Replace yourname.jpg with the image name you would like to use. Your Image will be displayed on the left hand of the screen and the text on the right. If you would like the text to be on the left hand side and the image on the right please change the word left in the source code to right.
 <IMG BORDER="0" ALIGN="Left" SRC="yourimage.jpg"> Your Text
No Example
Removing the underlines on links. Replace the http://www.yourdomain.com with your webpage you wish to link too and then replace your link with what you wish the link to display.
 <A HREF="http://www.yourdomain.com" STYLE="TEXT-DECORATION: NONE">Your Link</A>
No Example
Changing the color of a link. First, change http://www.yourdomain.com to where you would like the link to go. Then change #FF0000 to the color you would like in hexa-decimals.
 <A HREF="http://www.yourdomain.com"><FONT COLOR="#FF0000">Your Link</FONT></A>
Home Grown Productions 
You can display a description of your link in the status bar of Internet Explorer when your mouse is placed over the link.
<A HREF="http://www.yourdomain.com" onMouseOver="window.status='Your text description'; return true" onMouseOut="window.status='';return true">Your linked text</a>
Home Grown Productions 
Add transitions to your website just like in the movies. Just add the single line code between the <head> </head> tags of a page. To change the length of the transition change the number after Duration= and then to change weather u want a fade or mix or whatever change the number after Transition=. There are 23 diffrent transitions so choose a number from 0 to 22 and if you hit 23 that is a random transition.
 <META http-equiv="Page-Enter" CONTENT="RevealTrans(Duration=4,Transition=0)">
No Example
To open a link in a new window. Change http://www.yourdomain.com/ to your web page you would like to go to and then change Your Link to what you would like the link to be displayed as
 <A HREF="http://www.yourdomain.com/" TARGET="_blank">Your Link</a>
Home Grown Productions  
If you have an old web address and would automatically like your browser to forward anyone who goes to an old web address to a new web address place. Change http://www.newlocation.com/page.html to your new webaddress. You will place this source code on your old website.
<META HTTP-EQUIV="Refresh" CONTENT="0; URL=http://www.newlocation.com/page.html">
